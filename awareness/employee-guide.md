# Employee Guide for Responsible AI Usage

## Introduction

Artificial intelligence can improve productivity and efficiency when used responsibly.

Employees remain accountable for all content, decisions and actions supported by AI systems.

## Before Using AI

- Verify data classification
- Verify confidentiality requirements
- Ensure compliance with internal policies

## Never Share

- Patient records
- Personal data
- Passwords
- API keys
- Internal confidential documents

## Validate Outputs

Always:

- Verify accuracy
- Verify relevance
- Verify compliance
- Verify sources when applicable

## Report Concerns

Report immediately:

- Suspected data exposure
- Incorrect outputs causing business impact
- Unauthorized AI usage
- Security concerns

## Key Principle

AI assists humans. Humans remain responsible for decisions.
