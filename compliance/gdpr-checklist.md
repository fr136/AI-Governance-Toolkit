# GDPR Checklist for AI Projects

## Governance
- [ ] Data processing purpose identified
- [ ] Legal basis documented
- [ ] Data controller identified

## Personal Data
- [ ] Personal data inventory completed
- [ ] Data minimization applied
- [ ] Retention period defined

## Security
- [ ] Access controls implemented
- [ ] Logging enabled
- [ ] Incident reporting process documented

## Rights of Individuals
- [ ] Access requests supported
- [ ] Rectification requests supported
- [ ] Erasure process defined

## Third Parties
- [ ] Supplier assessment completed
- [ ] Data processing agreements reviewed

## Review
- [ ] Annual compliance review planned
