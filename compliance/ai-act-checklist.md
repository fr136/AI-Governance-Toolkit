# EU AI Act Compliance Checklist

## Governance

- [ ] AI inventory established
- [ ] AI owner identified
- [ ] Risk assessment completed
- [ ] Human oversight defined

## Transparency

- [ ] Users informed when interacting with AI
- [ ] AI-generated content identifiable
- [ ] Documentation maintained

## Data Governance

- [ ] Data quality controls implemented
- [ ] Data sources documented
- [ ] Personal data reviewed

## Security

- [ ] Access management implemented
- [ ] Logging enabled
- [ ] Incident response process documented

## Human Oversight

- [ ] Human review process defined
- [ ] Escalation process documented
- [ ] Accountability assigned

## Monitoring

- [ ] Performance monitored
- [ ] Periodic review scheduled
- [ ] Corrective actions documented
